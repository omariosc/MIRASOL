# MIRASOL Workshop | MICCAI 2026

Official website for the **MIRASOL** (Medical Image Computing in Resource Constrained Settings) Workshop at [MICCAI 2026](https://conferences.miccai.org/2026/), Strasbourg, France.

**Live site:** [mirasol.rise-miccai.org](https://mirasol.rise-miccai.org/)

## About

MIRASOL connects researchers, medical experts, policy makers, regulators, and industry partners working on translational machine learning solutions for real-world medical imaging in resource-constrained settings, with a focus on underrepresented communities. The workshop is organised in partnership with [RISE-MICCAI](https://rise-miccai.org/).

## Development

Built with React, Vite, and React Router. Hosted on GitHub Pages.

```bash
npm install
npm run dev
```

## Deployment

Pushes to `main` automatically deploy via GitHub Actions to GitHub Pages.

```bash
npm run build
```
